                        Guia de Sobrevivência
                    
    
                
    

                
    
                    
    
                        
    O que é Linha de Comando (CLI)?
    

                        
    
                            A maioria das pessoas usa o computador clicando em ícones, botões e menus. Isso se chama
                            Interface Gráfica.
                        
    

                        
    
                            A Linha de Comando é outra forma de conversar com o computador: através do terminal, você
                            escreve uma instrução em texto, aperta Enter, e o computador executa.
                        
    

                        
    Pense como se fosse dar ordens muito claras, uma linha por vez.
    
                    
    

                    
    
                        
    Um exemplo simples
    

                        
    Se você montar os blocos para gerar o comando abaixo:
    

                        
    mkdir projetos
    

                        
    
                            Você está dizendo ao computador:
                            
    
                             "Crie uma pasta (mkdir) com o nome projetos" .
                        
    
                    
    

                    
    
                        
    Como um comando é estruturado
    

                        
    A maioria dos comandos segue uma lógica parecida com uma frase:
    

                        
    COMANDO + OPÇÕES + ALVO
    

                        
      
                            
    • Comando: a ação principal (ex: ls, mkdir).
      • 
                            
    • 
                                Opções: modificadores do comando, geralmente com um traço (ex: -l,
                                -r).
                            
      • 
                            
    • 
                                Alvo: em quem ou onde a ação acontece (ex: pasta_fotos,
                                arquivo.txt).
                            
      • 
                        
    
                    
    

                    
    
                        
    🧩 O que é o ShellBlocks?
    

                        
    
                            O ShellBlocks é um tradutor visual. Ele ajuda você a aprender linha de comando sem precisar
                            decorar tudo de uma vez.
                        
    

                        
      
                            
    • Blocos: representam comandos, opções e alvos.
      • 
                            
    • Encaixar blocos: significa montar uma instrução válida.
      • 
                            
    • Código Gerado: mostra o comando real que você montou.
      • 
                            
    • Executar: roda o comando em um terminal simulado.
      • 
                        
    
                    
    

                    
    
                        
    🎮 Sistema de níveis
    

                        
    A ferramenta funciona como um jogo:
    

                        
      
                            
    • Cada nível apresenta um desafio
      • 
                            
    • Você monta o comando usando blocos
      • 
                            
    • Se fizer certo, desbloqueia o próximo
      • 
                        
    

                        
    Cada nível ensina um conceito novo, passo a passo.
    
                    
    

                    
    
                        
    💡 Dicas de Ouro
    
                        
      
                            
    • 
                                Maiúsculas importam:
                                no Linux, Pasta é diferente de pasta.
                            
      • 
                            
    • 
                                A ordem importa:
                                o computador lê os comandos exatamente como você monta.
                            
      • 
                            
    • 
                                Leia a missão:
                                ela diz exatamente o que se espera de você.
